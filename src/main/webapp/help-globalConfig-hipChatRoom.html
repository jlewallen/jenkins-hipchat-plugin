Enter the name of the room to which notifications should be sent. Note that this can be either the name of the room OR the id number, e.g. "Dev Team", and that multiple names may appear comma separated. While names are more readable, room numbers will not change over time and are therefore more resilient. Use your admin token to get a list of room ids here.

You can customize the room name per-project, but should always enter a default here.

The HipChat Message Api is used to send the message.
